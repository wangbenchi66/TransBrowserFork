# TransBrowser
一款可以透明化的浏览网页应用

## 使用说明

1. 运行程序
2. 右键右下角图标设置
3. 输入要浏览的网页
4. 调整大小、透明度

## 新功能（v2.0）

### A. 鼠标穿透模式（Click-through）
- 按 **Ctrl+Alt+T**（可自定义）切换穿透/可交互状态
- 穿透状态下所有鼠标事件透传到下层窗口/桌面
- 切换后系统托盘气泡提示当前状态
- 托盘菜单"鼠标穿透模式"勾选项同步状态（可点击切换）

### B. 全局快捷键（可自定义）

| 快捷键 | 操作 |
| --- | --- |
| **Alt+Q** | 老板键：快速隐藏/显示窗口 |
| **Ctrl+Alt+↑** | 增加不透明度（+5%，范围 20%–100%） |
| **Ctrl+Alt+↓** | 减少不透明度（–5%，范围 20%–100%） |
| **Ctrl+Alt+T** | 切换鼠标穿透模式 |
| Alt+D | 显示/隐藏（旧版快捷键，保留兼容） |
| Alt+F | 窗口置顶（旧版，保留兼容） |
| Alt+Shift+Z | 后退（旧版，保留兼容） |
| Alt+Shift+C | 执行 JS（旧版，保留兼容） |

**自定义快捷键**：打开"设置"→"全局快捷键设置"，点击对应输入框后按下新组合键，点击"应用快捷键"保存。配置持久化重启后生效。冲突时给出提示。

### C. 多 Tab 支持
- 标签栏显示在窗口顶部，点击 **"+"** 标签新建标签页
- 每个标签有独立的 WebView2 实例，URL、导航历史互不干扰
- 标签标题自动更新为网页标题
- **鼠标中键** 或 **右键 → 关闭标签** 关闭当前标签（至少保留一个）
- 链接默认在新标签中打开

### D. 自定义右键菜单
在网页内容区域右键弹出菜单，包含：
- **复制链接**：仅在右键点击链接时可用，自动写入剪贴板
- **开发者工具**：打开 WebView2 DevTools 调试面板
- 保留浏览器默认菜单项

### E. 系统托盘图标
- 最小化或点击关闭（X）自动缩小到系统托盘，气泡提示恢复方式
- 托盘图标**双击**恢复窗口
- 托盘右键菜单包含：
  - 显示 / 隐藏
  - 鼠标穿透模式（带勾选状态）
  - 设置
  - 退出

### Bug 修复：窗口缩放四角拖动
- 修复了**左下角**和**右下角**无法拖动缩放的问题
- 通过 `WM_NCHITTEST` 消息处理正确返回 `HTBOTTOMLEFT` / `HTBOTTOMRIGHT` / `HTBOTTOM`
- 四个角（左上/右上/左下/右下）及四条边均可拖动缩放

## 手动验证步骤

### 1. 窗口缩放四角验证
1. 启动程序，将窗口调整到适中大小
2. 将鼠标移到**左下角**，光标应变为↙形，拖动可缩放 ✓
3. 将鼠标移到**右下角**，光标应变为↘形，拖动可缩放 ✓
4. 将鼠标移到**左上角**和**右上角**，同样可拖动缩放 ✓
5. 四条边（上下左右）拖动也应正常工作 ✓

### 2. 快捷键验证
1. 按 **Ctrl+Alt+↑** 确认透明度增加（托盘气泡不显示，但窗口透明度变化）
2. 按 **Ctrl+Alt+↓** 确认透明度减少
3. 按 **Alt+Q** 确认窗口隐藏；再次按 **Alt+Q** 确认窗口恢复显示
4. 打开设置，修改一个快捷键，点击"应用"，重启后验证新键生效

### 3. 鼠标穿透验证
1. 按 **Ctrl+Alt+T** 开启穿透模式，托盘气泡提示"鼠标穿透已开启"
2. 在窗口区域点击鼠标，应无法点到 TransBrowser（点击穿透到下层）
3. 再次按 **Ctrl+Alt+T** 关闭穿透，气泡提示"已关闭"，窗口恢复可交互
4. 通过托盘菜单"鼠标穿透模式"也可切换，勾选状态与实际状态同步

### 4. 托盘恢复验证
1. 点击任务栏最小化按钮，窗口消失并显示气泡提示
2. 在托盘图标上**双击**，窗口恢复
3. 在托盘图标右键菜单选择"显示 / 隐藏"，窗口隐藏/恢复

### 5. 多 Tab 验证
1. 点击标签栏中的 **"+"**，新建标签页并自动切换
2. 在新标签导航到不同网址，与第一个标签互不影响
3. 鼠标中键点击标签关闭它
4. 右键标签→"关闭标签"同样关闭
5. 只剩一个标签时，关闭操作无效（不会关闭最后一个）

### 6. 右键菜单验证
1. 在网页中找到一个链接，右键点击，菜单顶部出现"复制链接"
2. 点击"复制链接"，粘贴验证内容正确
3. 在网页空白处右键，"复制链接"项不出现
4. 右键菜单底部有"开发者工具"，点击打开 DevTools 面板
