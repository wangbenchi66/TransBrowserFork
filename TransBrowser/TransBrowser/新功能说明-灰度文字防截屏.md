# TransBrowser 新功能 - 灰度模式、文字缩放、防截屏

## 本次新增功能

### 1. ✅ 灰度模式（摸鱼必备）

**功能说明**：
将网页内容转换为灰度显示，适合在需要低调浏览时使用。

**特点**：
- 使用 CSS `filter: grayscale(100%)` 实现
- 应用于整个HTML元素，包括图片、视频、背景等
- 可与其他模式（无图模式、透明背景）同时使用

**使用方法**：
1. 打开设置窗口
2. 找到"灰度模式"开关
3. 开启后所有网页显示为黑白

**技术实现**：
```javascript
// 注入灰度滤镜CSS
html{filter:grayscale(100%)!important;-webkit-filter:grayscale(100%)!important}
```

---

### 2. ✅ 文字缩放（护眼必备）

**功能说明**：
调整网页文字大小，范围 50%-200%。

**特点**：
- 50%：缩小文字（适合查看更多内容）
- 100%：默认大小
- 200%：放大文字（适合护眼、老年人使用）
- 使用滑块实时调整
- 所有标签页同步应用

**使用方法**：
1. 打开设置窗口
2. 找到"文字缩放"滑块
3. 拖动滑块调整文字大小

**技术实现**：
```javascript
// 使用CSS calc()函数缩放字体
*{font-size:calc(1em * 缩放比例)!important}
```

---

### 3. ✅ 防截屏模式（隐私保护）

**功能说明**：
防止截屏软件捕获窗口内容，保护隐私。

**特点**：
- 使用 Windows API `SetWindowDisplayAffinity`
- 阻止大部分截屏软件（QQ截图、微信截图、Snipping Tool 等）
- 系统要求：Windows 10 2004 或更高版本
- 开启后会显示提示气泡

**使用方法**：
1. 打开设置窗口
2. 找到"防截屏"开关
3. 开启后截屏软件无法捕获此窗口

**注意事项**：
- ⚠️ 需要 Windows 10 2004 (Build 19041) 或更高版本
- ⚠️ 不支持系统自带的 Print Screen 键
- ⚠️ 某些专业截屏工具可能绕过保护
- ⚠️ 开启后可能影响屏幕录制软件

**技术实现**：
```csharp
// 使用 WDA_EXCLUDEFROMCAPTURE 标志
SetWindowDisplayAffinity(handle, 0x00000011);
```

**错误处理**：
- 如果系统版本不支持，会显示警告对话框
- 自动关闭开关，不影响正常使用

## 配置项说明

### Properties.Settings 新增项

#### GrayscaleMode（灰度模式）
- **类型**：Boolean
- **默认值**：False
- **作用**：控制是否启用灰度显示

#### TextScale（文字缩放）
- **类型**：Int32
- **默认值**：100
- **范围**：50-200
- **作用**：控制网页文字缩放比例

#### AntiScreenshotMode（防截屏模式）
- **类型**：Boolean
- **默认值**：False
- **作用**：控制是否启用防截屏保护

### App.config 新增项
```xml
<setting name="GrayscaleMode" serializeAs="String">
  <value>False</value>
</setting>
<setting name="TextScale" serializeAs="String">
  <value>100</value>
</setting>
<setting name="AntiScreenshotMode" serializeAs="String">
  <value>False</value>
</setting>
```

## 使用场景

### 灰度模式
1. **工作摸鱼**：浏览娱乐网站时看起来像在查资料
2. **低调阅读**：降低视觉冲击，适合公共场所
3. **护眼模式**：减少色彩刺激

### 文字缩放
1. **老年人使用**：放大文字到 150%-200%
2. **浏览长文**：缩小到 70%-80% 查看更多内容
3. **视力保护**：根据环境调整舒适的阅读大小

### 防截屏
1. **隐私保护**：防止他人截图窥探
2. **公司环境**：防止监控软件截屏
3. **敏感内容**：浏览需要保密的信息
