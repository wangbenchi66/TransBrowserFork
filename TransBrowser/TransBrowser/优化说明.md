# TransBrowser 优化说明

## 本次更新内容

### 1. ✅ 全新标签栏设计

**改进内容**：
- 去除固定宽度限制，标签宽度自适应内容
- 现代化视觉设计：
  - 选中标签：纯白背景 + 蓝色下划线指示器

- 更清晰的文字显示
- 统一的视觉语言，类似 Chrome/Edge 的标签样式

**视觉效果**：
- 选中标签有明显的蓝色下划线（#0078D7）
- 关闭按钮 X 符号在悬停时变为红色
- 标签之间只有细微的分隔线，无明显间距
- 整体更简洁、现代

### 2. ✅ INI 配置文件系统

**新增文件**：
- `config.ini` - 主配置文件，包含所有可配置项
- `Tools\IniConfigManager.cs` - 配置管理器

**主要特性**：
1. **详细的注释说明** - 每个配置项都有：
   - 用途说明
   - 可用值范围
   - 默认值
   - 示例

2. **自动验证** - 配置值会被验证：
   - 数字范围检查（如不透明度 1-100）
   - 颜色格式验证（支持颜色名和 #RRGGBB）
   - 坐标/尺寸合理性检查
   - 快捷键格式验证

3. **错误处理** - 如果配置值有问题：
   - 自动使用默认值
   - 启动时显示警告对话框
   - 列出所有错误的配置项
   - 不会导致程序崩溃

4. **手动编辑友好** - 可以直接用记事本编辑：
   ```ini
   [Window]
   Opacity=100        # 不透明度
   ThemeColor=#FF5733 # 使用十六进制颜色
   
   [Hotkeys]
   BossKey=Alt+Q      # 老板键
   ```

### 3. ✅ 配置项说明

#### 窗口外观 `[Window]`
- `Opacity` (1-100) - 窗口不透明度
- `Position` (X,Y) - 窗口位置
- `Size` (宽,高) - 窗口大小
- `ThemeColor` - 主题颜色（颜色名或 #RRGGBB）
- `NoTitle` - 是否隐藏标题栏
- `TopMost` - 是否置顶
- `ShowInTaskbar` - 是否显示在任务栏
- `HoverHeaderMode` - 标题栏悬停显示模式

#### 标签栏 `[TabBar]`
- `ShowTabBar` - 是否显示标签栏

#### 浏览器功能 `[Browser]`
- `DefaultUrl` - 默认打开网址
- `UserAgent` - 自定义 User-Agent
- `AutoHide` - 失去焦点时自动隐藏
- `NoImageMode` - 无图模式
- `TransparentBackground` - 网页背景透明
- `MobileMold` - 移动模式
- `ClickThroughMode` - 鼠标穿透模式

#### 快捷键 `[Hotkeys]`
- `BossKey` - 老板键（默认 Alt+Q）
- `OpacityUp` - 增加不透明度（默认 Alt+Right）
- `OpacityDown` - 减少不透明度（默认 Alt+Left）
- `OpacityReset` - 重置不透明度（默认 Alt+Up）
- `ClickThrough` - 切换鼠标穿透（默认 Ctrl+Alt+T）
- `GoBack` - 后退（默认 Alt+Shift+Z）
- `RunJs` - 运行 JS（默认 Alt+Shift+C）

### 4. ✅ 使用方法

#### 首次使用
1. 启动程序会自动生成 `config.ini`
2. 程序位于 `bin\Debug\config.ini` 或 `bin\Release\config.ini`
3. 所有设置都有默认值，可以直接使用

#### 自定义配置
1. 关闭程序
2. 用记事本打开 `config.ini`
3. 按照注释说明修改配置
4. 保存后重新启动程序

#### 配置示例
```ini
# 设置窗口半透明，黑色主题
[Window]
Opacity=70
ThemeColor=#2D2D30

# 自定义快捷键
[Hotkeys]
BossKey=Ctrl+Shift+H
OpacityUp=Ctrl+Plus
OpacityDown=Ctrl+Minus

# 开启无图模式
[Browser]
NoImageMode=true
```

### 5. ⚠️ 注意事项

1. **配置文件位置**：
   - 在程序所在目录（exe 文件旁边）
   - 不在项目源代码目录

2. **编辑配置前**：
   - 建议先关闭程序
   - 使用 UTF-8 编码保存（记事本默认即可）

3. **错误处理**：
   - 如果配置值格式错误，程序会提示并使用默认值
   - 不会导致程序无法启动
   - 可以删除 `config.ini` 让程序重新生成

4. **与界面设置的关系**：
   - 目前设置界面仍使用 `Properties.Settings`
   - INI 文件作为高级配置和备份
   - 未来版本可能完全迁移到 INI

5. **快捷键格式**：
   - 修饰键：Ctrl, Alt, Shift, Win
   - 可以组合：Ctrl+Alt+Q
   - 按键名：A-Z, 0-9, F1-F12, Up/Down/Left/Right 等

### 7. 📋 开发说明

#### IniConfigManager 使用示例
```csharp
// 创建配置管理器
var config = new IniConfigManager();

// 读取配置（自动验证）
int opacity = config.GetInt("Window", "Opacity", 100, min: 1, max: 100);
Color color = config.GetColor("Window", "ThemeColor", Color.White);
bool topMost = config.GetBool("Window", "TopMost", false);
string hotkey = config.GetHotkey("Hotkeys", "BossKey", "Alt+Q");

// 写入配置
config.SetInt("Window", "Opacity", 80);
config.SetColor("Window", "ThemeColor", Color.Black);
config.SetBool("Window", "TopMost", true);

// 检查错误
if (config.HasErrors)
{
    config.ShowErrors(); // 显示错误对话框
}
```

#### 添加新的配置项
1. 在 `config.ini` 中添加配置项和注释
2. 在代码中使用 `IniConfigManager` 读取
3. 添加验证逻辑（范围、格式等）
