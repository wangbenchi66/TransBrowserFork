# TransBrowser 快速使用指南

## 🎯 核心功能速览

### 摸鱼必备
| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 老板键 | `Alt+Q` | 立即隐藏/显示窗口 |
| 失焦隐藏 | - | 切换窗口自动隐藏 |
| 防截屏 | - | 防止监控软件截图 |
| 灰度模式 | - | 网页显示黑白，看起来正经 |
| 鼠标穿透 | `Ctrl+Alt+T` | 点击穿透窗口 |

### 显示调整
| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 提高透明 | `Alt+←` | 窗口更透明 |
| 降低透明 | `Alt+→` | 窗口更不透明 |
| 重置透明 | `Alt+↑` | 恢复完全不透明 |
| 文字缩放 | - | 50%-200% 调整字体 |

### 网页功能
| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 无图模式 | - | 不显示图片和视频 |
| 透明背景 | - | 网页背景透明 |
| 灰度模式 | - | 网页黑白显示 |

## 🚀 快速开始

### 1. 摸鱼场景
```
步骤：
1. 设置 → 开启"失焦隐藏"
2. 设置 → 开启"灰度模式"
3. 设置 → 开启"防截屏"
4. 浏览你想看的网页

当老板来了：
- 按 Alt+Q 立即隐藏
- 或者切换到其他窗口（自动隐藏）
```

### 2. 阅读场景
```
步骤：
1. 设置 → "文字缩放" 滑到 150%
2. 设置 → 开启"灰度模式"（可选，护眼）
3. 设置 → 开启"透明背景"（可选，降低亮度）

享受舒适的阅读体验！
```

### 3. 隐私保护
```
步骤：
1. 设置 → 开启"防截屏"
2. 设置 → 开启"无图模式"（加载更快）
3. 按 Alt+Q 设置老板键

你的隐私得到保护！
```

## ⚙️ 设置说明

### 基础设置
- **透明度**：0% = 完全不透明，100% = 几乎透明
- **状态栏背景色**：自定义窗口顶部颜色
- **失焦隐藏**：窗口失去焦点自动隐藏

### 标签栏设置
- **显示标签栏**：多标签浏览
- **任务栏悬停显示**：鼠标移到顶部显示标题栏

### 网页设置
- **无图模式**：隐藏所有图片和视频
- **网页背景透明**：强制网页背景透明
- **灰度模式**：网页黑白显示（新）
- **文字缩放**：调整网页文字大小 50%-200%（新）

### 安全设置
- **防截屏**：防止截屏软件捕获（新）

## 💡 使用技巧

### 技巧1：组合拳
```
灰度模式 + 无图模式 + 防截屏 = 极致摸鱼
```

### 技巧2：快捷操作
```
Alt+Q：紧急隐藏
Alt+←→：调整透明度
Alt+↑：快速恢复不透明
```

### 技巧3：新标签
```
设置窗口点击"打开"按钮 → 自动在新标签打开网页
点击顶部的"+"号 → 快速新建标签
中键点击标签 → 快速关闭标签
```

### 技巧4：自定义网站
```
新标签页可以添加常用网站
点击"+"添加，点击"×"删除
```

## 🔧 常见问题

### Q1: 防截屏不工作？
A: 需要 Windows 10 2004 或更高版本。检查系统版本：
```
Win + R → winver → 确认版本号 ≥ 19041
```

### Q2: 老板键失效？
A: 检查是否有其他软件占用了 Alt+Q。可以在设置中更改快捷键。

### Q3: 文字缩放没效果？
A: 某些网站使用绝对像素值，尝试调整到 150% 以上。

### Q4: 网页透明无效？
A: 部分网站（如微信读书）需要一些时间加载，等待2-3秒。

### Q5: 灰度模式只对部分内容生效？
A: 刷新页面后生效。某些动态加载的内容需要重新应用。
